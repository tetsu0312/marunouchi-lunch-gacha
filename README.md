# marunouchi-lunch-gacha
今日のお昼どうしようって迷ったことがあったので、自分用にランチガチャを作りました。
ボタンを押すと、ランチ候補をランダムで表示します。

## 🔗 サイト
https://tetsu0312.github.io/marunouchi-lunch-gacha/

## 🛠 使っているもの
- HTML / CSS / JavaScript
- GitHub Pages（デプロイ用）

## 📁 ファイル構成
- index.html：トップページ
- about.html：このサイトについて
- style.css：スタイル
- script.js：ガチャの処理
- lunch_list.json：ランチデータ
